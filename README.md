# repl

## What is it ?
Simple repl for lisp expressions done in node.js.
Will improve to support full programming language features in the future.


## How to use it ? 

```npm run repl```

```sh
repl> (+ 1 2)
3
```

```sh
repl> (+ 1 (- 10 (* 1 (/ 6 2))))
8
```

## How to run tests ?
```npm test```
